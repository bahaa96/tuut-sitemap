# TUUT Shop Sitemap CDN

This repository contains the sitemap.xml file for TUUT Shop, automatically uploaded by the sitemap generation script.

## Purpose

- Host sitemap.xml on GitHub Pages (free CDN)
- Accessible at: https://tuut.shop/sitemap.xml
- Automatically updated by `npm run generate-sitemap`

## Setup

1. Create GitHub repository: `tuut-shop/tuut-sitemap`
2. Enable GitHub Pages (Settings → Pages → Deploy from branch → main/root)
3. The upload script will automatically commit and push updates

## Files

- `sitemap.xml` - Generated sitemap with all site URLs
- Auto-generated by the main tuut-website project

## CDN URL

Once GitHub Pages is enabled, the sitemap will be available at:
- Direct: https://tuut-shop.github.io/tuut-sitemap/sitemap.xml
- Proxied: https://tuut.shop/sitemap.xml (via Next.js)
